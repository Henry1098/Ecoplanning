A - Installation du pilote de la clef de protection



Le programme d'installation de la clef est rédigé en anglais, mais le principe est le même que dans tout programme d'installation
Termes utilisés:
Cancel = Annuler
Next = Suivant
Close = Fermer
Finish = Terminer
Setup = Installation
Driver = Pilote
InstallShield : c'est le nom du programme qui fait l'installation

Avant tout, retirez la clef de protection de votre ordinateur.
Ecran de lancement : c'est l'écran qu'on obtient en lançant le programmme SETUP.EXE (ce lancement est automatique)


Traductions:

Veuillez attendre pendant que l'assistant InstallShield prépare le setup
Le pilote iKey prépare l'assistant InstallShield, qui va vous guider pendant le processus d'installation. Veuillez Attendre.
